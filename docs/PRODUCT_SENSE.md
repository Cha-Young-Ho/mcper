# docs/PRODUCT_SENSE.md — 제품 철학

---

## 제품 비전

**MCPER**: 에이전트(Cursor/Claude) 중심의 기획서·규칙·스킬 관리 플랫폼

### 핵심 가치

1. **에이전트 자율성** — MCP 도구로 정보 검색·규칙 수정, 독립적 의사결정
2. **감시 추적** — Append-only 버전 기록, 누가/언제/무엇을 변경했는지 추적
3. **간결함** — 최소 의존성, 한눈에 이해되는 구조

---

## 사용자 페르소나

| 사용자 | 목표 | 핵심 필요 |
|--------|------|----------|
| 개발자 (에이전트) | 빠른 컨텍스트 검색 + 규칙 준수 | MCP 검색, 규칙 조회, 코드 분석 |
| 시니어/PM | 기획서·규칙 관리 + 일관성 유지 | 어드민 UI, 버전 이력, MCP 통계 |
| 운영자 (DevOps) | 안정적 인프라 운영 | 헬스체크, 모니터링, 자동 확장 |

---

## 경쟁 우위

- **MCP 네이티브** — 에이전트가 직접 도구 호출 (API 래퍼 불필요)
- **Append-only 규칙** — 버전 충돌 없음, 롤백 가능
- **자체 호스팅** — 프라이버시 보장, 오프라인 가능
- **코드 그래프** — CodeNode 기반 영향도 분석
