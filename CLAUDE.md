## 공통 규칙 (필수)

1. **컨텍스트 절약** — 파일 읽기 전 추측 금지, 변경 전 확인, 요청 없는 개선 금지, compact 응답
2. **문서화** — 큰 작업 끝나면 `docs/dev_log.md` 에 append (날짜/에이전트/요약/파일/결정사항)
3. **선호 추적** — 파악된 스타일/선호는 `.agents/*/` 의 해당 파일 갱신
4. **협업** — 모호한 요청은 확인 후 진행, 선행 작업은 `dev_log.md` 최신 항목 먼저 읽기

---

## 협업 흐름

```
@pm (타당성) → @planner (기획) → @senior (설계) → @coder (구현) → @infra (검수)
```

단순 작업은 중간 단계 생략 가능. 예: 버그픽스는 `@coder` 만.
